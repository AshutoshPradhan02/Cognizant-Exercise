# Cognizant Upskilling Program - Frontend Training Repository

A focused training repository for **Module 1** frontend topics. The current module includes Bootstrap, CSS3, HTML5, JavaScript, and jQuery exercise sets.

## What's Included

- **Module 1: Bootstrap Exercises** - Bootstrap 5 topic pages covering setup, grid, utilities, components, icons, JavaScript plugins, and Sass customization
- **Module 1: CSS3 Exercises** - 11 CSS3 exercises covering styling, layout, responsive design, and debugging
- **Module 1: HTML5 Exercises** - 10 HTML5 exercises for a Local Community Event Portal with forms, media, storage, geolocation, and DevTools practice
- **Module 1: JavaScript Exercises** - 14 JavaScript exercises covering data, functions, objects, arrays, DOM rendering, events, async logic, forms, fetch, debugging, and jQuery comparison
- **Module 1: jQuery Exercises** - 6 jQuery exercises covering library inclusion, selectors, methods, DOM manipulation, and event helpers
- **Professional Documentation** - Theory, examples, and best practices for each exercise
- **Responsive Design** - Mobile-first approach and multi-device testing
- **External Stylesheets** - Organized, commented CSS for real-world practice
- **Developer Tools Guide** - Learn to debug and test like a professional

## Installation

### For Bootstrap Exercises

```powershell
cd "d:\Campus\Cognizant\Cognizant-Exercise\Upskilling\Bootstrap-Exercises"
npm install
npm run build:css
```

### For CSS3 Exercises

No installation needed - all exercises are standalone HTML files.

### For HTML5, JavaScript, and jQuery Exercises

No installation needed for normal browsing. Open each topic `index.html` in Chrome.

For the JavaScript Fetch examples, a local server is recommended because some browsers restrict loading `data/events.json` from `file://`.

## How to use

### Starting Point
Open `Upskilling\index.html` in your browser to see all available modules and exercises.

### Module Structure
- Each module has an `index.html` hub page listing all exercises
- Click any exercise to start
- Read the theory, study the examples, and complete the challenges
- Use Chrome DevTools (F12) to inspect and debug
