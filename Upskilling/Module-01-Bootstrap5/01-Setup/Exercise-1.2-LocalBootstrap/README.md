# Local Bootstrap Setup

This exercise demonstrates using Bootstrap 5 with files downloaded directly into the project.

## Exercise solved

- 1.2: Set up a project using npm or downloaded Bootstrap files and use those files in a sample HTML page.

## How it is structured

- `bootstrap/css/bootstrap.min.css` — compiled Bootstrap CSS used by the page.
- `bootstrap/js/bootstrap.bundle.min.js` — compiled Bootstrap JavaScript bundle including Popper.

## Why local setup is useful

A local setup is useful when you need offline development, full control over asset versions, or to deploy without external CDNs.

## How to run

Open `index.html` in your browser. The page uses local Bootstrap CSS and JavaScript files.
