Bootstrap Sass Customization Exercises

Exercises solved: 19.1, 19.2.

This page explains npm Sass setup and Bootstrap theme customization using variable overrides.

Exercise 19.1: npm and Sass setup

Install Bootstrap, Bootstrap Icons, and Sass using npm to enable a production-style build process.

npm install bootstrap@5.3.7 bootstrap-icons sass --save-dev

The `package.json` in this folder includes scripts to build and watch Sass files.

Exercise 19.2: Customize primary colors and border radius

Override Bootstrap variables in `scss/_variables.scss` before importing Bootstrap.

Custom theme preview

The compiled CSS file `css/custom.css` includes the updated primary color and border radius used by this page.
