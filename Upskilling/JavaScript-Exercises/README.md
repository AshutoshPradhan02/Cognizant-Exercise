# JavaScript Exercises - Local Community Event Portal

Module 1 topic folder for JavaScript exercises.

## Included

The `index.html`, `css/styles.css`, `js/main.js`, and `data/events.json` files cover all 14 tasks:

- External JavaScript setup
- Variables, data types, template literals, and operators
- Conditions, loops, and error handling
- Functions, closures, callbacks, and higher-order filtering
- Objects, classes, prototypes, and `Object.entries()`
- Array methods including `push()`, `filter()`, and `map()`
- DOM manipulation and event card rendering
- Click, change, and keydown event handling
- Fetch with `.then()`, `.catch()`, and `async/await`
- ES6+ features
- Form handling and inline validation
- AJAX-style POST submission with delayed response simulation
- Console and Network debugging hooks
- A note comparing jQuery with modern frameworks

Open `index.html` in a browser. Some Fetch examples work best through a local server because browsers may restrict local JSON loading from `file://`.
