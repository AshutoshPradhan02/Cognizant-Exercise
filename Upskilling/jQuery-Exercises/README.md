# jQuery Exercises

Module 1 topic folder for jQuery exercises.

## Included

1. jQuery inclusion by CDN and local fallback
2. `$()` selector usage with `$(document).ready()`
3. Common methods: `.hide()`, `.show()`, `.fadeOut()`, `.fadeIn()`, `.toggle()`, and chained `.slideUp().delay().slideDown()`
4. DOM manipulation with form input, new `li` creation, append, and remove all
5. Click and double-click event handling for color changes
6. Event helpers for click, double-click, mouse enter, mouse leave, and keypress

Open `index.html` in a browser. The CDN version of jQuery is preferred, and `vendor/jquery-local.js` is available as an offline teaching fallback.
