Exercise 4: Color & Background Styling

Scenario

You're theming the event portal based on a city council's branding. You need to apply consistent colors and create visually appealing backgrounds.

Objective

Apply consistent colors, gradients, and backgrounds to create a professional, themed appearance for the portal.

Color Formats

1. Hex Color Format (#RRGGBB)

Six-digit hexadecimal color representation. Most common in web design.

color: #667eea; /* blue */
color: #FF0000; /* red */
color: #00FF00; /* green */
This box uses HEX color #E74C3C (red) for background and #FFFFFF (white) for text

2. RGB Format

Red, Green, Blue values (0-255)

color: rgb(102, 126, 234); /* blue */
color: rgb(255, 0, 0); /* red */

3. RGBA Format (with Transparency)

RGB with alpha channel (0-1 for opacity)

background-color: rgba(102, 126, 234, 0.2); /* 20% opacity */
background-color: rgba(102, 126, 234, 0.5); /* 50% opacity */
background-color: rgba(102, 126, 234, 1); /* fully opaque */
This box uses RGBA with 20% opacity for background and full opacity for border

Background Properties

Background Color

body {
  background-color: #f9f9f9;
}

Linear Gradient (top to bottom)

background: linear-gradient(180deg, #667eea 0%, #764ba2 100%);
Linear Gradient (Top to Bottom)

Linear Gradient (diagonal)

background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
Linear Gradient (Diagonal - 135deg)

Radial Gradient

background: radial-gradient(circle, #667eea 0%, #764ba2 100%);
Radial Gradient (Circular)

Background Image with Fallback

background-image: url('images/pattern.png');
background-color: #f0f4ff; /* fallback */
background-size: cover;
background-position: center;
Background Image with Color Fallback. (SVG pattern + fallback color)

Background Properties Summary

Property Description Example
background-color Sets solid background color #667eea
background-image Sets background image or gradient url('image.jpg') or linear-gradient(...)
background-size Controls image size cover, contain, 100px 50px
background-position Positions background image center, top left, 50% 50%
background-attachment Image scrolls or fixed scroll, fixed
background-repeat Repeats background image repeat, no-repeat, repeat-x

Best Practices

  • Always provide fallback colors for gradients and images
  • Ensure sufficient contrast between text and background for accessibility
  • Use RGBA for transparency when you need semi-transparent overlays
  • Keep color palette consistent across the entire site
  • Test colors on different devices as they may appear differently
  • Use gradients sparingly to maintain professional appearance

Challenge

Your Task: Create a landing page for the community event portal with:
• A hero section with a linear gradient background
• Different colored cards for different event categories
• RGBA colors for semi-transparent overlays
• A color scheme with primary, secondary, and accent colors
Ensure good contrast for readability!
