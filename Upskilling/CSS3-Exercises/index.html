Exercise Overview

Welcome to the CSS3 Exercises section! This collection of 11 exercises teaches you how to style a Local Community Event Portal using modern CSS techniques. Each exercise builds upon previous concepts to create a professional, responsive, and visually appealing website.

Theme: Local Community Event Portal

Imagine you're building a website for local community event announcements, registrations, and management. Through these exercises, you'll create a complete styling solution for this portal.

Learning Path

These exercises are designed to be completed sequentially. Each exercise builds upon the concepts from previous ones.

  1. Foundational (Exercises 1-3): Learn CSS inclusion methods and selector types.
  2. Styling Basics (Exercises 4-6): Master colors, typography, and link/list styling.
  3. Layout Fundamentals (Exercises 7-8): Understand tables and the box model.
  4. Advanced Features (Exercises 9-10): Create columns and responsive layouts.
  5. Professional Skills (Exercise 11): Debug and test your work like a pro.

Requirements & Tools

  • A modern web browser (Chrome, Firefox, Safari, or Edge)
  • A code editor (VS Code recommended)
  • Chrome DevTools for debugging and testing
  • Basic HTML knowledge
  • Internet connection for Google Fonts
