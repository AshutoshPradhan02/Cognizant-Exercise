Exercise 5: Typography - Fonts and Text

Scenario

The marketing team wants more appealing fonts and better readability for the event portal. Typography is key to creating a professional, engaging website.

Objective

Enhance textual appearance and readability using Google Fonts and CSS typography properties.

Font Selection

Using Google Fonts

Import Google Fonts in your HTML <head>:

<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Roboto:wght@400;700&display=swap" rel="stylesheet">

1. Serif Font (Playfair Display)

Great for headlines and elegant content

font-family: 'Playfair Display', serif;
This is a beautiful serif font for headlines

2. Sans-Serif Font (Roboto)

Perfect for body text and web content

font-family: 'Roboto', sans-serif;
This is a clean sans-serif font for body text and paragraphs. It's highly readable on screens.

Font Properties

Font Size

Control text size using px, rem, or em units

font-size: 16px; /* pixels */
font-size: 1rem; /* relative to root (16px) */
font-size: 1.2em; /* relative to parent */

This text is small (0.85rem)

This text is large (1.5rem)

Font Weight

Control text thickness (100-900, or keywords)

font-weight: 400; /* normal */
font-weight: 700; /* bold */
font-weight: normal|bold|lighter|bolder;

This text is light (300)

This text is bold (700)

Font Style

Make text italic or oblique

font-style: normal;
font-style: italic;
font-style: oblique;

This text is italic

This text is bold

Text Properties

Text Alignment

Align text horizontally

text-align: left|center|right|justify;

This text is center aligned

This text is right aligned

Text Transform

Change text case without modifying HTML

text-transform: uppercase|lowercase|capitalize;

this text is transformed to uppercase

this text is capitalized

Letter Spacing

Control space between characters

letter-spacing: 3px; /* positive for more space */
letter-spacing: -1px; /* negative for less space */

This text has increased letter spacing

Line Height

Control vertical space between lines

line-height: 1.6; /* unitless (multiplier) */
line-height: 20px; /* fixed pixels */
This paragraph has increased line-height (2), making it much more readable. Generous spacing between lines is important for body text. Many designers recommend 1.5 to 2 for optimal readability.

Word Spacing

Control space between words

word-spacing: 8px;

This text has increased word spacing between every word

Text Shadow

Add shadow effect to text

text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
/* offset-x | offset-y | blur-radius | color */

This text has a subtle shadow effect

Text Decoration

Underline, overline, or strikethrough

text-decoration: underline;
text-decoration-color: #667eea;
text-decoration-thickness: 3px;

This text has a custom underline

Typography Summary Table

Property Values Purpose
font-family Font name or stack Choose typeface
font-size px, rem, em, % Set text size
font-weight 100-900, normal, bold Text thickness
font-style normal, italic, oblique Italic or slanted
line-height Unitless, px, em Vertical space between lines
letter-spacing px, em Space between characters
word-spacing px, em Space between words
text-align left, center, right, justify Horizontal alignment
text-transform uppercase, lowercase, capitalize Change text case

Typography Best Practices

  • Limit fonts: Use 2-3 fonts maximum (1 for headings, 1 for body)
  • Font size: Body text should be 16px minimum for readability
  • Line height: Use 1.5-1.8 for body text for comfortable reading
  • Font pairing: Use serif + sans-serif or similar weight combinations
  • Contrast: Ensure sufficient contrast between text and background
  • Google Fonts: Use for better font selection and performance
  • Web-safe fonts: Always provide fallbacks in font-family stack

Challenge

Your Task: Design a page with professional typography:
• Import 2 Google Fonts
• Create a hierarchy: H1 (serif, large), H2 (sans-serif, medium), body (sans-serif)
• Set line-height to 1.6+ for body text
• Use letter-spacing for visual interest
• Test readability on different screen sizes
