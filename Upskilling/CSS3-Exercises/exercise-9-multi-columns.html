Exercise 9: Multiple Columns in Text

Scenario

The community bulletin needs to be displayed like a newspaper with multiple columns. CSS3 multi-column layout allows you to create newspaper-style text flow.

Objective

Use CSS3 multi-column layout properties to create professional newspaper-style text presentations.

Multi-Column CSS Properties

column-count

Specifies the number of columns to divide content into:

column-count: 2; /* divide into 2 columns */
column-count: 3; /* divide into 3 columns */
column-count: 4;

column-gap

Space between columns (like gutter in newspaper):

column-gap: 2rem; /* 2rem space between columns */
column-gap: 30px; /* 30 pixels */
column-gap: normal; /* browser default (usually 1em) */

column-rule

Visual separator line between columns (similar to border):

column-rule: 1px solid gray;
column-rule: 2px dashed #667eea;
column-rule: 3px dotted #764ba2;

2-Column Article Example

Community Newsletter - March 2024

Welcome to March's community newsletter! This month has been filled with exciting events and opportunities for our neighbors to connect. We're thrilled to share updates about recent activities and upcoming gatherings that will bring our community closer together.

The Community Potluck last month was a tremendous success with over 40 families attending. Everyone brought delicious dishes representing their cultural backgrounds, making it a truly diverse and enriching experience. Special thanks to the volunteers who helped set up and clean up. Mark your calendars for next month's event on April 19th!

Our local youth center has launched a new after-school program focusing on STEM education. High school students can now volunteer as mentors, earning community service hours while making a difference in the lives of younger children. Applications are open through March 31st.

Upcoming events this month include the Community Garden Workshop (March 18), Book Club Meeting (March 22), and the Spring Cleanup Initiative (March 25). Whether you're interested in gardening, reading, or environmental stewardship, there's something for everyone. Please check our website for times and locations.

3-Column Article Example

Extended News Article - Community Development

Our community has seen significant growth and positive changes over the past year. New businesses have opened downtown, creating local jobs and revitalizing our commercial district. The community development board has approved funding for street improvements and new public spaces.

The new community center, which opened in February, has already become a hub for activities. Classes range from yoga and fitness to art and music. Over 300 residents have already registered for spring classes. The center also hosts community meetings and social events on weekends.

Public transportation improvements are underway, with new bus routes planned for next summer. The city council approved funding for better bike lanes and pedestrian walkways. These infrastructure improvements will make our community more accessible and environmentally friendly.

Our schools continue to excel academically. This year's science fair attracted entries from over 200 students, showcasing incredible innovation and creativity. Several students have qualified for the state-level competition, representing our community with distinction.

Advanced Multi-Column Features

column-span

Allow an element to span across all columns (like a headline):

.multi-column h3 {
  column-span: all; /* spans across all columns */
}

This heading spans across all columns!

The heading above uses column-span: all to span across all columns. This is perfect for section headers or important announcements that should break the column flow and draw attention. The text below continues to flow in multiple columns as normal.

column-width (Responsive Columns)

Browser automatically determines number of columns based on available width:

column-count: auto;
column-width: 300px; /* each column is ~300px wide */
column-gap: 2rem;

Result: On a 1200px wide screen, you'd get ~4 columns. On a 600px screen, you'd get 2 columns.

Multi-Column Properties Summary

Property Values Purpose
column-count number | auto Fixed number of columns
column-width length | auto Minimum width of each column
columns Shorthand Set count and width together
column-gap length | normal Space between columns
column-rule width style color Divider line between columns
column-span none | all Span across all columns

Use Cases for Multi-Column Layout

  • News articles - Create newspaper-style layouts
  • Blog posts - Make long-form content more readable
  • Community newsletters - Professional publication style
  • Book-like content - Traditional reading experience
  • Case studies - Dense information in organized format

Responsive Multi-Column

Make multi-column responsive with media queries:

/* Desktop - 3 columns */
@media (min-width: 1024px) {
  .article {
    column-count: 3;
  }
}

/* Tablet - 2 columns */
@media (min-width: 768px) and (max-width: 1023px) {
  .article {
    column-count: 2;
  }
}

/* Mobile - 1 column */
@media (max-width: 767px) {
  .article {
    column-count: 1;
  }
}

Browser Compatibility

Multi-column layout is well-supported in all modern browsers:

  • Chrome: Supported since 2012
  • Firefox: Supported since 2011
  • Safari: Supported since 2009
  • Edge: Fully supported
  • IE 10+: Partial support
