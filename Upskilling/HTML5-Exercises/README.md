# HTML5 Exercises - Local Community Event Portal

Module 1 topic folder for the HTML5 exercise set.

## Included

1. HTML5 base template with semantic comments
2. Navigation and section linking
3. Welcome banner using ID, class, internal CSS, and inline styles
4. Image gallery using a table and image attributes
5. Event registration form with validation and output
6. Event handling for blur, change, click, double-click, and keyboard events
7. Video invite using `video` and `oncanplay`
8. User preferences with `localStorage` and `sessionStorage`
9. Geolocation with high accuracy options and error handling
10. DevTools debugging notes and console logging
