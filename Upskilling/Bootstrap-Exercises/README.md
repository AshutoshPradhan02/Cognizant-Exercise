# Bootstrap 5 Exercises

This folder contains a professional and interview-ready implementation of the Bootstrap 5 exercise solutions.

## What is included

- `index.html` � central exercise hub with links to every solution page.
- `setup.html` � Exercise 1.1, 1.2, 2.1, 2.2: Bootstrap CDN setup, local setup, folder structure, and Bootstrap JS bundle.
- `grid-layout.html` � Exercise 3.1, 3.2, 4.1, 4.2, 5.1, 5.2: responsive grid system, column layouts, sidebar, centering, and reordering.
- `flexbox.html` � Exercise 6.1, 6.2: responsive flexbox utilities for navigation and card layout alignment.
- `typography.html` � Exercise 7.1, 7.2: typography utilities and text case utilities.
- `forms.html` � Exercise 8.1, 8.2: Bootstrap form controls, input groups, checkboxes, and floating labels.
- `buttons.html` � Exercise 9.1, 9.2: contextual buttons, button groups, and toggle buttons.
- `navigation.html` � Exercise 10.1, 10.2: responsive navbar, search form, tabs, and pills.
- `cards-media.html` � Exercise 11.1, 11.2: profile card and media object layout.
- `utilities.html` � Exercise 12.1 through 16.2: spacing, pricing layout, backgrounds, visibility, borders, shadows, and positioning.
- `icons-js.html` � Exercise 17.1 through 18.2: Bootstrap Icons, icon-only buttons, modal, and accordion.
- `sass-customization.html` � Exercise 19.1, 19.2: Sass setup and variable customization.
